== README

This is a single page Ruby on Rails app using the Pokeapi (http://pokeapi.co/)
Pokeapi is currently on v2 but this application uses v1.


REQUIREMENTS
-RAILS 4.2.4
-RUBY 2.2.3

After you clone the application run 'bundle install' to install gem file. This application does not use a database.
Rails s to start development sever.
Enjoy
